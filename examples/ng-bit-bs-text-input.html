ng-bit-bs-text-input

This component inserts a Bootstrap text input with label. It needs Bootstrap CSS loaded to be properly viewed.

Example

{{name}}

Source

<ng-bit-bs-text-input
    label="Your Name"
    model="name"
    placeholder="Put your name in here">
</ng-bit-bs-text-input>
<span>{{name}}</span>

Parameters

  • model: string - Model that will be bind for the input.
  • type: string - The input type: text, email, password. (default: text)
  • label: string - The label that will be shown. Optional.
  • placeholder: string - The placeholder that will be shown inside the input. Optional.
  • name: string - Name for the input. Optional.
  • minlength: string - HTML minlength attribute. Optional.
  • maxlength: string - HTML maxlength attribute. Optional.
  • input-id: string - The id for input element. Optional.
  • ng-required: Angular directive for required. Optional.
  • ng-minlength: Angular directive for min length. Optional.
  • ng-maxlength: Angular directive for max length. Optional.
  • ng-pattern: Angular directive for pattern. Optional.
  • ng-change: Angular directive for change. Optional.
  • ng-trim: Angular directive for trim. Optional.
