ng-bit-threejs-panorama

This component can be used to create a equirectangular panorama visualization using the three.js library.

Example

Panorama picture by Mantis of Destiny.

Source

<ng-bit-threejs-panorama
    picture="assets/panorama.png">
</ng-bit-threejs-panorama>

Parameters

  • picture: string - Equirectangular panorama image.
  • library-url: string - The URL for the three.js library to be automatically loaded when auto-load is true. Optional.
  • width: integer - Width of the panorama viewer (default: 500)
  • height: integer - Height of the panorama viewer. (default: 300)
  • auto-load: boolean - Automatically loads the three.js library (default: true)
  • auto-spin: boolean - Automatically spins the panorama. (default: true)
  • speed: integer - The speed for the auto-spin. (default: 1)
  • draggable: boolean - Makes the panorama visualization draggable. (default: true)
  • disable-zoom: boolean -Disables the mouse wheel zooming. (default: false)
